Hospedagem de Sites

Passos para hospedagem

  • Criar um repositório no remoto (GitHub)
  • Criar repositório Local: Comando git init
  • Adicionar arquivos em zona de preparação: git add .
  • Criar ponto na história: git commit -m "first commit"
  • Renomear a branch para main: git branch -M main
  • Unificar repositórios: git remote add origin [endereço-repositórios github]
  • Enviar o código: git push -u origin main
